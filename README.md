## Modelagem e Treinamento da Rede Neural

O modelo de Rede Neural Feed-Forward (FFNN) foi implementado utilizando Python com TensorFlow/Keras e Scikit-learn. A seguir, os principais aspectos da modelagem:

**Arquitetura do Modelo**
    
- Camada de Entrada: 7 neur√¥nios, representando as vari√°veis clim√°ticas:

    - Evapora√ß√£o (evap)
    - Insola√ß√£o (inso)
    - Precipita√ß√£o (prec)
    - Temperatura (temp)
    - Umidade (umid)
    - Velocidade do Vento (vent)
    - Data (em formato Unix)

- Camada Oculta: 1 camada com 666 neur√¥nios e fun√ß√£o de ativa√ß√£o ReLU.

- Camada de Sa√≠da: 1 neur√¥nio para a previs√£o do volume √∫til (volu).

**Configura√ß√µes de Treinamento**

- Divis√£o dos Dados:
    - Treino: 75%
    - Teste: 25% (Utiliza-se train_test_split do Scikit-learn com random_state = 42.)

- Normaliza√ß√£o: 

    As vari√°veis de entrada (X) foram normalizadas via StandardScaler. As vari√°veis de sa√≠da (y) foram mantidas em seu formato original para representar valores reais.

- Par√¢metros do Treinamento:
    - Otimizador: Nadam com learning_rate de 0.0003.
    - Fun√ß√£o de Perda: Erro Quadr√°tico M√©dio (MSE).
    - Early Stopping: Monitoramento de val_loss com paci√™ncia de 21 √©pocas, interrompendo o treinamento aos 694 ciclos dentro de um m√°ximo de 2500 √©pocas.

O trecho abaixo resume a configura√ß√£o e o treinamento:

```python

# Configura√ß√£o do EarlyStopping

                       callbacks=[parar_rede_neural], verbose=1)
```
